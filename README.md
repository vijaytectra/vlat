# VLAT Exam Web Application

A responsive web application for VLAT (Vinayaka Mission's Law Admission Test) built with HTML, Tailwind CSS, and JavaScript.

## Pages

1. Home
2. About VLAT
3. About VMLS
4. Blog Page
5. Announcements (Sidebar popup - available on all pages)
6. How to Register
7. VLAT Exam Structure
8. Contact Us
9. Login page
10. Forgot password
11. Check Your Email
12. Reset password
13. Success page
14. Welcome screen
15. Student Dashboard (with different navbar)
16. Mock Test Instruction screen
17. Test screen
18. Jump to Question popup and Submit popup
19. Score Board screen

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Design System

See `design-system.md` for color variables, typography, and component specifications extracted from Figma.

## Features

- Fully responsive design (mobile-first)
- Tailwind CSS for styling
- Interactive JavaScript components
- Announcements sidebar (accessible from all pages)
- Form validation
- Question navigator for test screens
- Timer functionality for exams
