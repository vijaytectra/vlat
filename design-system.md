## Typography

- Font Family: Inter (system fallback)
- Responsive text sizing using Tailwind CSS utilities

## Layout

- Container max-width: 1440px (desktop)
- Mobile-first responsive design
- Breakpoints: sm (640px), md (768px), lg (1024px), xl (1280px), 2xl (1536px)

## Components

- Buttons: Primary (red), Secondary (outlined)
- Forms: Input fields, textareas, checkboxes
- Navigation: Header with logo, menu items, and user profile
- Cards: For content sections, test cards, blog posts
- Modals/Popups: For announcements, jump to question, submit confirmation
